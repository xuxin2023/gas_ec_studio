# 更新日志

## 0.1.0 RC4 - 2026-07-18

### 修复

- 修复正式主题下 EC 处理方法控制台首行参数在常见桌面视口中被滚动区域裁切的问题。
- 修复连续 UI 测试中采集事件循环未关闭，以及离屏 Qt 延迟销毁可能导致 Windows 原生崩溃的问题。
- 删除被重复定义覆盖的旧对标驾驶舱实现，消除不可达代码中的未定义变量。

### 改进

- 进入不确定度步骤时自动将首个可操作方法参数定位到视口内，同时保留标题、运行条和方法摘要。
- 清理结果导出清单和完整输出中的重复字段定义，保持现有导出值与字段顺序稳定。
- UI 几何回归显式加载正式主题，并以受控事件泵验证高频刷新合并行为。

### 验证

- 全部 `816` 项测试按四段完成，结果为 `319 + 106 + 210 + 181 passed`。
- 结果导出、平面拟合库和交付包重点回归 `10 passed`。
- 对标驾驶舱和报告中心定向回归 `5 passed`。

### 发布说明

- 当前为 Windows 10/11 64 位候选版本。
- 用户数据默认保存在 `%LOCALAPPDATA%\GasECStudio\runtime_data`。
- 当前候选包未进行代码签名；正式发布前仍需完成签名与时间戳验证。

## 0.1.0 RC3 - 2026-07-18

### 修复

- 修复设备数据持续更新时主窗口、设备中心和实时采集页出现闪烁的问题。
- 修复隐藏页面仍反复刷新，以及设备卡片在每个数据帧到达时被销毁并重建的问题。

### 改进

- 高频设备信号按 250 ms 窗口合并，低频视图切换和设备选择仍立即生效。
- 文本、日志和动态样式仅在内容真正变化时更新，减少无效布局和重绘。
- 右侧上下文检查器降低实时刷新频率，同时保留页面切换和操作后的即时状态。

### 验证

- 65 次连续设备更新由 65 次整页刷新收敛为 16 次可见页刷新。
- 同一受控样本中，绘制事件由 7,154 次降至 234 次，样式变化由 5,365 次降至 0 次。
- 设备详情和实时采集页在隐藏状态均不执行高频刷新。

### 发布说明

- 当前为 Windows 10/11 64 位候选版本。
- 用户数据默认保存在 `%LOCALAPPDATA%\GasECStudio\runtime_data`。
- 当前候选包未进行代码签名；正式发布前仍需完成签名与时间戳验证。

## 0.1.0 RC1 - 2026-07-06

- 提供设备接入、实时采集、项目与站点、EC 处理、谱修正与 QC、报告交付六个连续工作区。
