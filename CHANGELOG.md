# 更新日志

## 0.1.0 RC2 - 2026-07-18

### 新增

- 增加正式应用图标，并接入窗口、任务栏和 Windows 可执行文件。
- 增加应用内“关于 / 使用说明 / 更新日志”入口。
- 增加可维护的用户指南和版本更新日志，并随发布包一起提供。

### 改进

- 优化标题区品牌布局，在常见桌面视口保持工作区高度和控件边界稳定。
- 更新项目说明和候选包说明，使功能范围、数据位置和签名状态更清晰。

### 发布说明

- 当前为 Windows 10/11 64 位候选版本。
- 用户数据默认保存在 `%LOCALAPPDATA%\GasECStudio\runtime_data`。
- 当前候选包未进行代码签名；正式发布前仍需完成签名与时间戳验证。

## 0.1.0 RC1 - 2026-07-06

### 新增

- 提供设备接入、实时采集、项目与站点、EC 处理、谱修正与 QC、报告交付六个连续工作区。
- 支持报告、证据包、清单和交付包导出，并保留可审计的运行状态。

### 改进

- 优化报告与结果导出的重型构建缓存，减少同一进程内重复计算。
- 完善 1366 x 768 到 1600 x 900 桌面视口的自适应布局。
- 收紧发布界面和交付文件中的公开文本，保持产品表述独立一致。

### 发布说明

- 当前为 Windows 10/11 64 位候选版本。
- 用户数据默认保存在 `%LOCALAPPDATA%\GasECStudio\runtime_data`。
- 当前候选包未进行代码签名；正式发布前仍需完成签名与时间戳验证。
